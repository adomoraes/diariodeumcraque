Arquitetura da Solução e Stack de Tecnologia1. ArquiteturaAdotaremos uma arquitetura de serviços desacoplada, com um backend servindo uma API RESTful para um frontend Single-Page Application (SPA).Frontend (Cliente): Um PWA construído em React. Será responsável por toda a interface do usuário, gerenciamento de estado local e lógica de apresentação.Backend (Servidor): Uma API Node.js monolítica (para o MVP) que gerenciará a lógica de negócio, autenticação e comunicação com o banco de dados.Banco de Dados: Um banco de dados relacional para garantir a integridade e a estruturação dos dados.2. Stack de Tecnologia (Versões Estáveis ~2025)Linguagem Principal: TypeScript em todo o projeto (frontend e backend).BackendRuntime: Node.js v22 LTSFramework: NestJS 11+ORM: Prisma 5+Autenticação: JWT com Passport.jsFrontendBiblioteca: React 19+Build Tool: Vite 5+Styling: Tailwind CSS 4+Gerenciamento de Estado: ZustandRoteamento: React Router v6Banco de DadosProdução: PostgreSQL 16Desenvolvimento: Docker com imagem do PostgreSQL.DeploymentFrontend (PWA): VercelBackend & Banco de Dados: Render
